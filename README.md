# Quote Forge

Quote Forge is an offline-first quote follow-up app for tradespeople. It helps
manage customers, jobs, quotes, reminders, templates, attachments, and exported
PDFs while keeping business records on the user's device.

The current release does not use a hosted sync service, advertising SDKs, or
analytics SDKs. Customer, job, quote, settings, and attachment data are stored
locally unless the user chooses to create an encrypted backup file.

## Privacy Policy

The privacy policy explains what data the app processes, where that data is
